## 概要

従来の単語帳といえば、自身で単語と意味をペアで作成したり、提供されている単語帳を使用することでした。

このアプリは、翻訳を調べた単語が自動的に登録され、自分だけの単語帳を従来の手間を省きながら作成することができます。

また、忘却曲線を参考にしたリマインダー・テスト機能により単語帳以上の役割を果たすことができます。

## 特徴

特徴として以下の効果が期待できる。

- 単語の翻訳・復習の習慣化

- 復習の継続
